# 프로젝트 소개
SAND WITCH는 샌드위치를 좋아하지 않는 사람에게는 관심을 주고, 샌드위치를 좋아하는 사람에게는 좀 더 재미있고 맛있는 경험을 선사할 수 있도록 하는 서비스입니다.
맛을 더하는 "알고 먹기"를 위한 샌드위치의 탄생 역사부터, 세상 맛있는 200여가지의 샌드위치가 하늘에서 떨어집니다. 전국 어디서나 맛있는 샌드위치를 즐길 수 있는 전국 샌드위치 매장이 캐러셀로 돌아가고 있고, 나눌수록 맛있는 샌드위치를 나눈 인스타의 많은 사람들도 있습니다. 마지막으로 자신만의 개성있는 샌드위치를 직접 만들 수 있는 샌드위치 만들기까지 이런 다양한 인터렉션으로 누구나 즐겁게 샌드위치에 접할 수 있도록 하는게 SAND WITCH입니다.

# 주요 기능
## 체험을 즐겁게 하는 다양한 인터렉션
- 하늘에서 샌드위치가 떨어지고, 화면 어디서든 "order"를 입력하면 샌드위치가 나오는 등 다양한 인터렉션이 존재하여 사용을 즐겁게 할 수 있습니다.

## 직접 만들어보는 자신만의 개성있는 샌드위치
- 햄, 토마토, 치즈, 달걀, 새우, 상추의 재료로 만드는 자신만의 개성있는 샌드위치를 만들 수 있습니다.
